# Extension Icons

This directory should contain the following icon files:

- icon16.png (16x16 pixels)
- icon32.png (32x32 pixels) 
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

For now, you can use any placeholder icons or create simple ones. The extension will work without these icons but they improve the user experience in Chrome.

You can:
1. Create simple PNG icons with a music/social theme
2. Use online icon generators
3. Use placeholder images temporarily

The icons should represent the Tidal Social brand - perhaps a combination of music notes and social/connection symbols.
